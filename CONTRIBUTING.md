# Contributing to this FHIR IG

Thank you for considering contributing! Your input helps improve this guide for the community.

## How to Contribute
- **Report Issues**: Use GitHub Issues to report bugs or suggest improvements.
- **Propose Changes**:
  - Fork the repository.
  - Create a branch: `git checkout -b feature/your-change`
  - Commit and push your changes.
  - Submit a Pull Request.

## Coding Standards
- Follow HL7 FHIR IG Publisher conventions.
- Validate your IG locally before submitting.
